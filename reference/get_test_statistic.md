# Just in case anyone wants to invert a p-value to recover the chi-squared distributed test statistic

Just in case anyone wants to invert a p-value to recover the chi-squared
distributed test statistic

## Usage

``` r
get_test_statistic(pvalue, df)
```

## Arguments

- pvalue:

  the p-value to invert

- df:

  the degrees of freedom for the test
